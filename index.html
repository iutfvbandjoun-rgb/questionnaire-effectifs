📊 Exporter vers Google Sheets

Copiez les données ci-dessous et importez-les dans Google Sheets en quelques clics.

  1. Cliquez sur "Copier CSV" ci-dessous.
  2. Ouvrez sheets.new pour créer une nouvelle feuille Google.
  3. Cliquez sur une cellule vide, puis Ctrl+V (ou Cmd+V) pour coller.
  4. Dans le menu : Données → Fractionner le texte en colonnes, choisir Virgule comme séparateur.
  5. Vos données apparaîtront en colonnes IPES / FILIÈRE / VILLE / EFFECTIF.
