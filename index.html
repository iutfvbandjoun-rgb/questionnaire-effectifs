Étape 1 sur 4
Quel est l'IPES ?
Tapez les premières lettres pour filtrer. Si votre IPES n'est pas dans la liste, saisissez-le directement.
Étape 2 sur 4
Quelles villes ?
Sélectionnez une ou plusieurs villes où cet IPES est présent. Cliquez sur ✕ pour retirer une ville.
Étape 3 sur 4
Quelles filières ?
Sélectionnez toutes les filières dispensées dans cet IPES. Cliquez sur ✕ pour retirer une filière.
Étape 4 sur 4
Saisir les effectifs
Pour chaque filière, indiquez l'effectif d'étudiants dans chaque ville sélectionnée. Utilisez les boutons − / + ou tapez directement.
